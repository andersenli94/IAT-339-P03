Round Apocalypse - Mobile Game

PROFILE

Round Apocalypse is a top down scrolling shooter game created using Android Studio. The player travels in space in search for a habital planet while being hunted by their predators. The game utilizes the device's sensors for interactive gameplay keeps high scores using a SQLite database. I created the interface for Round Apocalypse and coded the functionality of the program.
